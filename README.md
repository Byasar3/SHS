# Self-Help Station: 

### Project aims:
1. to build a functioning app based off of the proposed wireframe
2. to eventually go live

### What is Self-Help Station?
In a nutshell it is a free mental-health web app that will offer self-help tools.

Self-help station is a place where you can come in times of need, to track your mental health and learn more about your mental health.

In the journey that is life, we need to stop the constant moving from place A to place B, we need to take a break, we need to check in with ourselves. Hence the name, self-help station
